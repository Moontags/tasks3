Build Register Number


function buildRegisterNumber(theLetters, theDigits) {
    const lettersRegex = /^[A-VYZ]{2,3}$/;
    const digitsRegex = /^(?:[1-9]\d{0,2}|0)$/; 

    if (!lettersRegex.test(theLetters)) {
        throw new Error("Invalid register number letters");
    }

    if (!digitsRegex.test(theDigits)) {
        throw new Error("Invalid register number digits");
    }

    return `${theLetters}-${theDigits}`;
}

function testRegisterNumber(letters, digits) {
    try {
        const registerNumber = buildRegisterNumber(letters, digits);
        console.log(`Valid register number: ${registerNumber}`);
        document.getElementById('result7').textContent = `Valid register number: ${registerNumber}`;
    } catch (error) {
        console.error(`Error: ${error.message}`);
        document.getElementById('result7').textContent = `Error: ${error.message}`;
    }
}

The buildRegisterNumber function takes two arguments, theLetters and theDigits. It uses regular expressions to validate both inputs.

  • The lettersRegex /^[A-VYZ]{2,3}$/ checks if the letters part consists of 2 to 3 uppercase letters from A to V or Y to Z (excluding W). The ^ and $ anchors ensure that the entire string must match the pattern.
  • The digitsRegex /^(?:[1-9]\d{0,2}|0)$/ checks if the digits part consists of one to three digits with no leading zeros (e.g., "1", "23", "100") or is the single digit "0". The non-capturing group (?:...) is used here.
  • If either theLetters or theDigits fails the respective regular expression test, the function throws an Error with a specific message.
  • If both arguments are valid, the function returns the combined register number in the format "letters-digits".
  • The testRegisterNumber function demonstrates how to use buildRegisterNumber and handles potential errors using a try...catch block, displaying the result or the error message on the page and in the console.
